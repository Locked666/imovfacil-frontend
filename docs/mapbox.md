# Mapbox

## Preparação

- Token via `VITE_MAPBOX_TOKEN`
- Mapa central na homepage
- Pins com GeoJSON source
- Clusterização nativa do Mapbox
- Sincronização entre lista e viewport
- Hook customizado em `src/hooks/use-map.ts`

## Observação

Sem token válido, o frontend mostra um estado explicativo e mantém o layout funcional.
